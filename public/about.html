Project Experience:

One of my significant projects, "Smart Forest Management," involved developing an IoT-based system to detect and prevent forest fires and theft. In this project, I integrated sensors such as humidity, temperature, and pressure with Arduino and NodeMCU. Using OpenCV, I implemented human activity detection to enhance security measures. The data collected by these sensors was sent to ThingSpeak via NodeMCU's WiFi module, enabling forest workers to manage the forest more efficiently. This project allowed me to apply my technical skills in a real-world context, enhancing my understanding of IoT and data integration. Another notable project is the "Smart Asset Locator," an IoT-based solution for efficient object tracking and retrieval. My role in this project involved integrating RFID technology. The system used RFID tags attached to valuables and a centralized RFID receiver to track objects within designated areas. The mobile app provided real-time feedback and sound alerts, making it easier for users to locate their items. This project further enhanced my skills in RFID integration and IoT.
